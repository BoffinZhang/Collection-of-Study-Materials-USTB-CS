### 主要内容
**自下到顶**介绍计算机网路各层的功能、主要协议。
有课内配套实验
### 考核要点
切记，掌握各主要协议不能只停留在表面概念，一定要**完全理解各个协议是如何发送报文、如何交换信息，路由表如何更新的**，要真正理解各个协议的运作机制
1. 物理层
    - 这部分考的不多
2. 链路层
   - PPP
   - 以太网
3. 网络层
   - IP
   - ICMP
   - 路由选择协议RIP（这个重点考）、OSPF、BGP（这俩没考）
4. 运输层
   - UDP
   - TCP
5. 应用层
   - DNS
   - HTTP
